# SpeakEase Gateway Web

本项目为 SpeakEase Gateway 的前端界面，基于 Vue 3、TypeScript 和 Vite 构建，参考主站前端架构，适用于 API 网关管理、监控等场景。

## 技术栈
- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Ant Design Vue & Element Plus
- Axios

## 快速开始

```bash
pnpm install
pnpm dev
```

## 目录结构
详见 src 目录，结构与主站前端一致。
